RedHead

Sistema desenvolvido para ser um sistema de gestão de uma empresa de desenvolvimento de software. Sistema desenvolvido em JavaFX com banco de dados MySQL.

O projeto como todo teve a duração de um semestre acadêmico. Seu objetivo era o aprendizado de desenvolvimento de sofwtare utilizando programação orientada a objetos, técnicas para gerenciamento de equipe, cronogramas de desenvolvimento, entregas em forma de sprint.
